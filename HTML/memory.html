Hier kann Memory für eine Person gespielt werden. Die Karten können können durch Anklicken umgedreht werden. Ein Klick auf "Regeln" zeigt die Spielregeln an. Mit "Neues Spiel" wird ein neues zufälliges Spielfeld erzeugt. "Lösung anzeigen" deckt alle Karten auf und beendet das aktuelle Spiel. Die Zahl über dem "Neues Spiel"-Knopf zeigt die Anzahl der vergangenen Züge an.

  • Es werden pro Zug jeweils zwei Karten umgedreht.
  • Bei einem ungleichen Paar werden die Karten wieder umgedreht.
  • Bei einem paar gleicher Karten bleibt das Paar aufgedeckt liegen.
  • Die Anordnung der Karten ändert sich während des Spiels nicht.
  • Das Spiel ist gewonnen, wenn alle Karten aufedeckt wurden.
